## Installation 
 * Clone this repo using
``git clone https://github.com/Yuma-Tsushima07/netscan.git``

 * First use 
 ``pkg update && pkg upgrade``.
* Install nmap using 
 ``pkg install nmap``.
* Finally run the script using
``bash scan.sh``
